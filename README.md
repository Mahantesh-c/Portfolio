- Smooth Scrolling Navigation

## ⚙️ How to Run Locally

1. Clone the repository
```
git clone https://github.com/Mahantesh-c/portfolio.git
```

2. Navigate to the project folder
```
cd portfolio
```

3. Open the project  
Open `index.html` in your browser.
